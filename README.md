# Economics of AGI

Research and modeling on the economics of transformative AI (TAI) — revenue models,
AI labour supply, and an accompanying brief/report.

## Contents

| Path | What it is |
|------|------------|
| `AI_revenue_model*.xlsx` | Revenue model spreadsheets (various iterations) |
| `tai_model/` | Core model: Python code, parameters, notebook, and rendered report |
| `tai_model/report.qmd` / `report.html` | The Quarto report (source + rendered) |
| `brief-draft.qmd` | Working draft of the brief |
| `*.docx` | Report drafts and reviews |
| `*.py` | Analysis and reproduction scripts |

## Collaborating

This repo is for **independent work synced via Git**, not live simultaneous editing.

- ⚠️ **Excel (`.xlsx`) and Word (`.docx`) files are binary** — Git cannot merge them.
  Coordinate so only one person edits a given spreadsheet/doc at a time, or split into
  separate files. The text/code files (`.py`, `.csv`, `.qmd`, `.md`) merge cleanly.
- Pull before you start, push when you finish a chunk: `git pull` → work → `git add -A`
  → `git commit -m "..."` → `git push`.
